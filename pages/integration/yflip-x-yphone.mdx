# YSeries | YFlip | YPhone

<Callout type="warning" emoji="⚠️">
    This requires YSeries | YFlip | YPhone 0.10.0 or higher
</Callout>
### Configuring

```lua
-- config/config.upload.lua
Config.Upload.Method = {
    Image = "Fivemanage",
    Video = "Fivemanage",
    Audio = "Fivemanage"
}
```

```lua
--server\apiKeys.lua
ApiKeys = {
    Image = "TOKEN_HERE",
    Video = "TOKEN_HERE",
    Audio = "TOKEN_HERE"
}
```

You can read more about configuring YSeries | YFlip | YPhone [here](https://docs.yseries.live/phone/configure#camera).
